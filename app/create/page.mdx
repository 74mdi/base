# Create

Spin up a new project with a simple structure you can expand later.

<Callout title='Template'>
  Keep only what you need on day one. Everything else can be added after the first review.
</Callout>

## Minimal Project Brief

- Name
- One-line goal
- Primary user
- Success metric
- First milestone

## First Week Actions

1. Collect references and constraints.
2. Decide scope boundaries.
3. Draft the first screen or flow.
4. Share a short update.

```md
# Project Name

## Audience
```
